# Sistem Informasi Perpustakaan

Simperpus adalah singkatan dari Sistem Informasi Manajemen Perpustakaan. Sistem Infromasi Manajamene adalah sebuah sistem atau perangkat lunak yang digunakan untuk mengelola berbagai aspek operasional perpustakaan.

## Fitur Utama

1. **Pendaftaran Anggota**
   - Pendaftaran anggota baru
   - Pembaruan data anggota
   - Penghapusan data anggota

2. **Katalog Buku**
   - Penginputan data buku
   - Pencarian katalog
   - Klasifikasi buku
   - Pengelolaan stok buku

3. **Sirkulasi Buku**
   - Peminjaman buku
   - Pengembalian buku
   - Perpanjangan masa pinjam

4. **Manajemen Inventaris**
   - Pemantauan kondisi fisik buku
   - Pencatatan buku hilang atau rusak
   - Pengadaan buku baru

5. **Laporan dan Statistik**
   - Jumlah peminjaman
   - Jumlah anggota aktif
   - Data statistik lainnya

6. **Manajemen Pengguna**
   - Pengaturan akses pengguna
   - Pengaturan hak akses dan peran pengguna

7. **Pengaturan dan Konfigurasi**
   - Konfigurasi notifikasi
   - Pengaturan denda keterlambatan
   - Pengaturan umum lainnya
